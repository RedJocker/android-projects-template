Description

In this stage, we will add the galleryButton as well as imageView to display image chosen from the gallery. The user should be able to load any image from his gallery to imageView.

The imageView is already set up in layout and MainActivity, so that it has a default bitmap from the start. Tests in later stages depend on this bitmap, so do not change this setup. In your virtual device you probably don't have any images saved in your gallery folder, so to make manual test to your application you can set up the gallery using the default camera app from your virtual device to add some pictures there


Objectives

  1. Add a Button for opening your gallery folder with the  id attribute btnGallery
  2. Add an ImageView which will display gallery image with the  id attribute ivPhoto(already done)
  3. Add OnClick method to your the galleryButton. It should launch new Intent like:
    Intent(Intent.ACTION_PICK, MediaStore.Images.Media.EXTERNAL_CONTENT_URI)
  4. Then you need to code what happens when this activity returns some result 

    resultLauncher = registerForActivityResult(StartActivityForResult()) { result ->
      if (result.resultCode == Activity.RESULT_OK) { ... }
    }

    That is mainly setting the URI of ivPhoto to the returned URI so that the image that was picked on gallery displays on our app
