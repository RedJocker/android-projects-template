Description

In this stage, we will implement our first filter — the brightness filter.

Image brightness is simply the average RGB value, so:

\[brightness = \frac {(Red + Green + Blue)} 3\]

To increase/decrease it, simply add or subtract the same amount from each RGB color. We will add a slider to adjust the image brightness.

Objectives

  • Add a Slider widget with the id attribute of slBrightness for specifying the photo brightness; the values may vary from -250 to 250 with the step size of 10 and the initial value of 0;
  • Attach onChangeListener to this slider that will apply the brightness filter;
  • The brightness filter should add the current value of the brightness slider to each RGB value of the default photo bitmap and check whether the new value is within the correct boundaries (0, 255);
  • Every time users load a new image, you must save its default bitmap and operate on it while applying a brightness filter (not on the current bitmap, because the filter will erase image details).

[ALERT-warning]Tests allow some margin error because of the possible differences that may occur due to rounding. Try to use Ints as much as possible, otherwise use Doubles.[/ALERT]
